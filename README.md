# 3D-Human-Skeleton-Anatomy-Tutor
3D Human Skeleton Anatomy Tutor — це інтерактивний веб-додаток, створений для студентів-медиків та всіх, хто цікавиться анатомією людини. Проект дозволяє досліджувати 3D-модель скелета, вивчати назви кісток українською та латиною, а також перевіряти свої знання у режимі іспиту.
